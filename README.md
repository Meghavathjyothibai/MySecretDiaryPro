# 📖 MySecretDiaryPro

**MySecretDiaryPro** is a full-stack digital diary app to **securely store personal diary entries, voice notes, images, and mood tracking**.  
Keep your thoughts organized and private with rich features and an interactive calendar view.

---

## ✨ Features

- User authentication (signup/login)
- Create, edit, and delete diary entries
- Attach images and voice notes to entries
- Track moods for each entry
- View entries in calendar format
- Search diary entries
- Export entries as PDF
- Daily writing streak tracker
- Cloud storage with MongoDB

---

## 🛠 Technologies Used

- **Frontend:** React.js, HTML5, CSS3, JavaScript, Framer Motion, React Quill  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Authentication:** JWT (JSON Web Token)  
- **Other Tools:** Axios, Cloudinary (image uploads)
